daitum-model
============

The ``daitum_model`` package provides a fluent builder API for defining the structure of a
Daitum optimisation model: tables, fields, named values (calculations and parameters), and
formula functions.

.. code-block:: python

   import daitum_model
   import daitum_model.formulas
