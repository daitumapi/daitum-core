Layout Views
============

Layout views and helper types for arranging elements on screen.

These classes provide two layout systems:

- **Flex**: one-dimensional layout (row/column), good for responsive toolbars, forms, and simple arrangements.
- **Grid**: two-dimensional layout (rows/columns), good for dashboards and tiled content.
