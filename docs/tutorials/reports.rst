Reports
=======

This tutorial covers how to define report views that surface optimisation results
to end users via the Daitum UI.

.. note::
   This tutorial is a work in progress. Full content coming soon.

Overview
--------

Reports are read-only views built with ``daitum_ui`` that present the outputs of
your optimisation model in a human-readable form.

Common report types:

- **Tabular views** — structured data in rows and columns
- **Gantt views** — timeline-based scheduling outputs
- **Form views** — summarised scalar outputs
