Templates & Versioning
======================

This tutorial covers how to structure your model code for reuse across projects
and how to manage model versions over time.

.. note::
   This tutorial is a work in progress. Full content coming soon.

Overview
--------

As models grow in complexity, managing shared logic and tracking changes becomes
increasingly important.

Key concepts:

- **Model templates** — reusable base configurations shared across model variants
- **Versioning strategy** — how to tag and release model packages
- **Changelogs** — communicating breaking changes to platform operators
