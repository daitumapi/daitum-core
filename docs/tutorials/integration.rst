Integration
===========

This tutorial explains how to connect your Daitum model to external data sources
and downstream systems.

.. note::
   This tutorial is a work in progress. Full content coming soon.

Overview
--------

Daitum models consume data from external systems and write results back to them.
Integration is configured through ``daitum_configuration`` using data source definitions.

Key concepts:

- **Data source types** — the kinds of systems Daitum can connect to
- **Input configuration** — mapping external data into the model
- **Output configuration** — writing solver results back to external systems
