daitum-core
===========

**daitum-core** is a collection of Python libraries for programmatically building
optimisation models for the Daitum platform.

.. code-block:: bash

   pip install daitum-core

This installs all three sub-packages:

- :doc:`daitum_model/index` — define tables, fields, formulas, and formula functions
- :doc:`daitum_ui/index` — define UI screens, navigation groups, and components
- :doc:`daitum_configuration/index` — build model configuration (algorithms, objectives, data sources)
