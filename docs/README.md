# daitum-core Documentation

Unified Sphinx documentation for `daitum-model`, `daitum-ui`, and `daitum-configuration` is
planned for a future release. See the second Jira ticket (MOD-133) for the documentation
consolidation plan.

The intended structure will be:

```
Getting Started
daitum-model
  ├── Tables
  ├── Fields
  ├── Named Values
  └── Formulas
daitum-ui
  ├── Views
  └── UI Actions
daitum-configuration
  ├── Algorithm
  ├── DVs
  ├── Objectives & Constraints
  ├── Processors
  ├── Reports
  └── Data Sources
```
